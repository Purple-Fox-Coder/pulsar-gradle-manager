# gradle-manager package

Manage gradlefrom inside Atom.
##### Commands
* (mac use control key)
* ctrl-k ctrl-t: toggles panel
* ctrl-k ctrl-k create new gradle pane
* ctrl-k ctrl-r: refreshes window
* ctrl-k ctrl-c: closes window
